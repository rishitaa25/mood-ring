## Overview

Mood Ring is an interactive embedded systems project that simulates the functionality of a traditional mood ring using an Arduino Uno and a temperature sensor. The system continuously measures the user's skin temperature and maps the reading to a predefined mood, which is then displayed through a custom-built website.

The project integrates embedded hardware, sensor interfacing, microcontroller programming, and web development to create a real-time visualization of the user's current temperature and corresponding mood.

This project provided hands-on experience with circuit design, embedded programming, hardware-software integration, and collaborative engineering design as one of my first computer engineering projects at Iowa State University.

---

## Project Objective

The objective of this project was to design and implement a temperature-based mood detection system capable of:

- Reading real-time temperature data from a sensor
- Processing sensor data using an Arduino Uno microcontroller
- Mapping temperature ranges to predefined moods
- Displaying the user's current mood and temperature on an interactive website
- Demonstrating communication between embedded hardware and a web application
